Feature: Boolean logic

Scenario: Numeric value equality
	Given a sheet with data:
	 | Row | A          |
	 |  1  | =<formula> |
	When the cell values are computed
	Then the values should equal:
	 | Cell | Value   |
	 |  A1  | <value> |

Examples:
	| formula | value |
	| 1=1     | True  |
	| (1=1)   | True  |
	| 1=0     | False |
	| 1+1=2   | True  |
	| 2=1+1   | True  |
	| 2-2=2   | False |
	| 3*3=9   | True  |
